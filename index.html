Grid Style Sheets

GSS replaces CSS layout & the browser’s layout engine by harnessing the Cassowary Constraint Solver — the same algorithm Apple uses to compute native layout.

Features

  • Constraint CSS
  • Grid-flavored VFL
  • Relative positioning & sizing
  • @if @else Blocks
  • Per-Element Responsiveness
  • Custom Constraint Variables
  • Accelerated 3D transforms *
  • New Pseudo Selectors
  • Nested Rules
  • Respects Text Re-Flows
  • (In)Equality Constraints
  • Constraint Strengths
  • Mutli-threaded Web Workers *
  • True Source Order Independence
  • Works with Normal CSS

* Maintains compatibility with browsers that don't support this feature
