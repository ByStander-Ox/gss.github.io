--- uid: CCSS title: GSS Reference subtitle: color: b layout: post --- ## Units of measurement The default unit of measurements is pixel. GSS also supports the following units: {% highlight css %} em /* use the font size of the element */ rem /* use the font size of the html element */ vh /* viewport height */ vw /* viewport width */ vmax /* the greatest value between vh or vw */ vmin /* the lowest value between vh or vw */ cm /* centimeter */ mm /* millimeter */ in /* inches */ {% endhighlight %}
